# Virachatbot
Sebuah project repositori Virachatbot, agen AI percakapan dinamis yang dikembangkan menggunakan express js,node.js dan integrasi API model bahasa canggih. Berfokus pada arsitektur sistem informasi yang aman dan efisien untuk kebutuhan informasi publik, manajemen FAQ otomatis, serta optimalisasi komunikasi digital pada lingkup instansi.
